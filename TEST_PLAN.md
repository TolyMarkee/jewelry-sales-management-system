## 十三、Postman API 测试脚本

```
### 1. 登录
POST http://localhost:8088/api/login
Content-Type: application/x-www-form-urlencoded

username=admin&password=123456

### 2. 获取个人信息（复制登录返回的 token）
GET http://localhost:8088/api/user/profile
Authorization: Bearer {{token}}

### 3. 商品列表
GET http://localhost:8088/api/product/list?page=1&limit=5
Authorization: Bearer {{token}}

### 4. 创建订单
POST http://localhost:8088/api/order/save
Content-Type: application/json
Authorization: Bearer {{token}}

{
  "customerId": 1,
  "totalAmount": 12597.00,
  "remark": "Postman测试订单",
  "items": [
    {"productId": 1, "quantity": 2, "price": 5999.00},
    {"productId": 8, "quantity": 1, "price": 599.00}
  ]
}

### 5. 入库
POST http://localhost:8088/api/stock/in
Content-Type: application/json
Authorization: Bearer {{token}}

{
  "productId": 1,
  "quantity": 10,
  "remark": "Postman测试入库"
}

### 6. AI 对话
POST http://localhost:8088/api/ai/chat
Content-Type: application/json
Authorization: Bearer {{token}}

{"message": "你好"}
```
